Class MemoryReservation

Namespace
MindControl
Assembly
MindControl.dll

Represents a reservation of a range of memory within a MemoryAllocation. Reservations within an allocation cannot overlap and thus can be used to safely manage data or code storage over a process. Disposing a reservation will free the memory range for other uses.

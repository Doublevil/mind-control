public enum HookRegister

Fields

Flags = 0

Represents the RFLAGS (in 64-bit) or EFLAGS (in 32-bit) register, holding CPU flags.

FpuStack = 41

Represents all ST registers (ST0-ST7). Since they work as a stack, the whole stack is saved and restored coordinatedly. These registers are not normally used. They are shared with the MMX registers (MM0-MM7). Do not use both at the same time.
